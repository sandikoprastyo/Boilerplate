Boilerplate with Webpack, Babel, Sass, React.

## Installation
```
git clone https://github.com/sandikoprastyo/Boilerplate.git
```
## Installation package run yarn or npm
```
yarn install
```
## Usage

### Development server

```bash
yarn start:dev
```

### Production build

```bash
yarn build
```
### Running server
[server running](http://localhost:8080/)
